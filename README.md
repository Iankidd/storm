## Spring Boot 通用基础框架
#### 涉及技术及工具：
 - 核心框架：spring boot 2.0.3 RELEASE
 - MVC 框架：spring MVC
 - ORM 框架：mybatis
 - AOP 框架：spring AOP
 - MyBatis 工具：mybatis pagehelper
 - 数据库：Mysql
 - 数据连接池：durid
 - 事务管理：spring transaction
 - 线程池管理：spring executor
 - 日志框架：logback
 - 定时任务：quartz
 - 缓存管理：redis
 - 安全权限管理：shiro
 - WEB 容器：tomcat
 - 模板引擎：thymeleaf
 - 后台模板框架：inspinia
