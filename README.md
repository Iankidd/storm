## Spring Boot 通用基础框架
#### 所用技术：
 - spring boot 2.0.3 RELEASE
 - spring MVC：MVC 框架
 - spring AOP：AOP 框架
 - spring transaction：事务管理框架
 - executor：spring 集成线程池框架
 - logback：日志框架
 - quartz：定时任务
 - redis：缓存框架
 - shiro：安全权限框架
 - mybatis：ORM 框架
 - durid：数据连接池
 - tomcat：WEB 容器
 - thymeleaf：模板引擎
 - inspinia：后台模板框架
